## 📌 Project Overview
This project was developed as the final evaluation for **Coding Skills 2** under the guidance of **Prakash Sir**. It serves as an interactive educational platform that allows users to encode text using the Huffman algorithm and understand the underlying data compression mechanics.

## 🚀 Features
- **Real-time Encoding**: Type text and instantly see it encoded using Huffman's algorithm.
- **Frequency Table Generation**: Analyzes input and generates the frequency distribution of symbols.
- **Huffman Tree Visualization**: Dynamically builds and displays the binary tree based on character frequencies.
- **Code Alphabet Mapping**: Shows the exact binary mapping assigned to each character.
- **Detailed Statistics**: Computes bit-length comparisons to demonstrate the compression efficiency over naive coding.

## 🛠️ Technology Stack
- **Framework**: [Next.js](https://nextjs.org/) (React)
- **Styling**: [TailwindCSS](https://tailwindcss.com/)
- **Visualizations**: `react-d3-tree` for rendering the Huffman tree structure.

## 💻 How to Run Locally
To run the project on your local machine:

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Open [http://localhost:3000](http://localhost:3000) in your browser.
